37@dataclasses.dataclass 

38class DIALConfig(MADQNConfig): 

39 """Configuration options for the DIAL system. 

40 

41 Args: 

42 environment_spec: description of the action and observation spaces etc. for 

43 each agent in the system. 

44 agent_net_keys: (dict, optional): specifies what network each agent uses. 

45 Defaults to {}. 

46 target_update_period: number of learner steps to perform before updating 

47 the target networks. 

48 executor_variable_update_period: the rate at which executors sync their 

49 paramters with the trainer. 

50 max_gradient_norm: value to specify the maximum clipping value for the gradient 

51 norm during optimization. 

52 min_replay_size: minimum replay size before updating. 

53 max_replay_size: maximum replay size. 

54 samples_per_insert: number of samples to take from replay for every insert 

55 that is made. 

56 prefetch_size: size to prefetch from replay. 

57 batch_size: batch size for updates. 

58 n_step: number of steps to include prior to boostrapping. 

59 sequence_length: recurrent sequence rollout length. 

60 period: consecutive starting points for overlapping rollouts across a sequence. 

61 discount: discount to use for TD updates. 

62 checkpoint: boolean to indicate whether to checkpoint models. 

63 optimizer: type of optimizer to use for updating the parameters of models. 

64 replay_table_name: string indicating what name to give the replay table. 

65 checkpoint_subpath: subdirectory specifying where to store checkpoints. 

66 learning_rate_scheduler_fn: function/class that takes in a trainer step t 

67 and returns the current learning rate. 

68 """
